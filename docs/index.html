LLVM Airfryer

An interactive CLI tool for managing LLVM builds and Compiler Explorer. Build LLVM, Zig, and run your own local Compiler Explorer — all from one menu.

Install LLVM Airfryer

click to copy curl --proto '=https' --tlsv1.2 -sSf https://raventid.github.io/llvm-airfryer/install.sh | sh
Copied to clipboard!

What does this do?

  • Detects your OS and architecture (Linux & macOS, x86_64 & ARM64)
  • Downloads the latest release binary from GitHub
  • Installs to $HOME/.llvm_airfryer/bin/llvm-airfryer
  • Creates an env file and updates your shell profile for PATH integration

After installation

Restart your terminal, or run:

. "$HOME/.llvm_airfryer/env"

Then launch the tool:

llvm-airfryer

Supported platforms

  • Linux x86_64 (GNU)
  • Linux ARM64 / aarch64 (GNU)
  • macOS x86_64 (Intel)
  • macOS ARM64 (Apple Silicon)
